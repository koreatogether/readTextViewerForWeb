# readTextViewer - WebView v3

`forWebView_03` 기준 버전입니다. 창 크기 변경 후에도 현재 보고 있는 문서 위치를 유지하고,
페이지 번호를 직접 입력해 원하는 페이지로 이동할 수 있습니다.

## 핵심 기능

- 스크롤 없는 고정 페이지 레이아웃
- 단면 / 양면 / 자동 전환
- 창 크기 변경 시 현재 위치 유지
- 페이지 번호 직접 입력 후 이동
- 마지막 페이지 및 읽던 위치 복원
- 검색, 북마크, 최근 파일, 드래그 & 드롭

## 사용 방법

1. `index.html` 을 브라우저에서 엽니다.
2. `파일 열기` 버튼으로 `.txt` 파일을 선택합니다.
3. 페이지 번호 입력칸에 숫자를 넣고 `이동` 버튼이나 `Enter`를 누르면 해당 페이지로 이동합니다.
4. 창 크기를 바꿔도 현재 보고 있는 위치가 유지됩니다.
